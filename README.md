# lite2d

#### 项目介绍
一款基于cocos2dx开发的文字游戏引擎。

#### 安装教程

1. 下载cocos2dx3.16解压并重命名为cocos2d放到根目录
2. 下载Resources解压到根目录
2. mac系统打开proj.ios_mac/avg-game.xcodeproj进行编译
3. windows系统安装vs2015打开proj.win32/avg-game.sln进行编译
4. android编译使用android studio打开proj.android-studio
